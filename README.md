# Catholic Faith Guardian - Professional Educational Website

## 📋 Project Overview

This is a complete rewrite of the Catholic theology educational website, now featuring a **polished, professional, modern design** that works beautifully on desktop, tablet, and mobile devices.

### 7. **Responsive Design**
- **Mobile-first CSS approach** (base styles for mobile, enhanced with breakpoints)
- **Breakpoints**:
  - **480px**: Mobile phones (small screens)
  - **768px**: Tablets and larger phones
  - **1024px+**: Desktop and large screens
- **Responsive typography**: Font sizes scale appropriately at each breakpoint
- **Flexible grid**: Cards adapt from single column (mobile) to 3 columns (desktop)
- **Touch-friendly**: Larger tap targets, adequate spacing
- **Tested on**: Mobile, tablet, and desktop views

### 8. **Typography & Readability**
- **Primary font**: Segoe UI (modern, professional, system font)
- **Font weights**: 300 (light), 600 (semi-bold), 700 (bold) for hierarchy
- **Line height**: 1.6-1.9 for optimal readability
- **Font sizes**: 
  - Hero title: 3.5rem (desktop) → 1.8rem (mobile)
  - Section titles: 2.8rem (desktop) → 1.8rem (mobile)
  - Body text: 1rem with good line spacing
  - Proper contrast ratios for accessibility

### 9. **Interactive Elements**
- **Hover effects** on cards (lift animation, color changes)
- **Button animations** (scale, shadow, color transitions)
- **Navigation animations** (hamburger menu transforms)
- **Smooth scrolling** throughout the site
- **Active state highlighting** on current page in navigation
- All animations use `transition: all 0.3s ease` for smoothness

### 10. **Code Quality**
- **Semantic HTML5**: Proper use of `<nav>`, `<main>`, `<section>`, `<footer>`
- **Clean CSS**: Well-organized with clear sections and comments
- **Professional JavaScript**: Modular, readable code for interactivity
- **Accessibility**: Proper heading hierarchy, alt text for images, semantic markup
- **No dependencies**: Pure HTML, CSS, and vanilla JavaScript (no frameworks needed)
- **Beginner-friendly**: Code is well-commented and easy to understand and modify

### 11. **Footer**
- **Professional design** with dark background
- **Responsive grid layout** that stacks on mobile
- **Three sections**:
  1. Site information and description
  2. Quick navigation links
  3. Topic shortcuts with anchor links

## 🔧 Customization Guide

### Change Colors
Edit `style.css` and update the color values:
- Primary: #1a472a
- Accent: #c9a961
- Background: #faf9f7

### Modify Content
Each page is standalone HTML with clear structure:
- Update hero sections in `<section class="hero">`
- Edit card content in `<div class="card">`
- Modify footer links in `<footer>`

### Add More Pages
1. Copy an existing page (e.g., christology.html)
2. Update the HTML content
3. Update navigation links to include new page
4. No CSS changes needed
