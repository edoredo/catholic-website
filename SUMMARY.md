## 🎨 DESIGN SYSTEM

### **Color Palette** (Professional Catholic-Inspired)
```

#f4f2ee - Light Grey (Sections)
#f9f7f3 - Subtle Highlight
#e8e6e0 - Light Border
```

### **Typography**
- Font: Segoe UI (system font - fast loading)
- Weights: 300 (light), 600 (semi-bold), 700 (bold)
- Sizes scale responsively: 1.8rem (mobile) → 3.5rem (desktop)
- Line height: 1.6-1.9 for optimal readability

### **Spacing**
- Section padding: 80px (desktop) → 50px (mobile)
- Card padding: 40px (desktop) → 20px (mobile)
- Gap between cards: 30px
- Grid-based: Uses `repeat(auto-fit, minmax(300px, 1fr))`

## 🚀 HOW TO USE

### **View the Website**
1. Open `index.html` in a browser
2. Click navigation links to explore
3. Test on mobile by resizing browser window
4. Hamburger menu appears at 768px and below

### **Modify Content**
1. Open any `.html` file in VS Code
2. Find the content you want to change (clearly labeled sections)
3. Edit text between tags
4. Save and refresh in browser

### **Change Colors**
1. Open `style.css`
2. Find color definitions near the top (clearly documented)
3. Change color hex values (e.g., `#c9a961` → your preferred color)
4. Save and refresh

### **Add a New Page**
1. Copy `christology.html` and rename it
2. Update the hero title and content sections
3. Update navigation links in all pages to include the new page
4. Link back to home in footer
